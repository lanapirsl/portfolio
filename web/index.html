02.04.01. responsive website

focus building a website with a responsive layout, meaning one that will adapt to whichever device we view it on to ensure quality user experience.

the design and content of the website show and tell of a specific art style - in this case, the bauhaus movement, taking into account its emphasis on functionality and simplicity.

starting with desktop view, i adjusted the content and layout of the website going to smaller devices.

learning in the first assignment of the basic web theme, i practiced content development, and HTML and css skills, as well as applying some new practices in the process of designing and creating a website.
